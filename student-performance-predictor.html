How It Works

The model uses Multiple Linear Regression — a supervised ML technique that finds the best-fit linear relationship between input features and the target (Final Score).


Training: The normal equation θ = (XᵀX)⁻¹ Xᵀy is solved analytically on the 500-record dataset using JavaScript.


Features: Study Hours, Attendance, Assignments, Previous Marks, and Class Participation are normalized before training.
